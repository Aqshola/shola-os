# Setup Guide

## Prerequisites

- Node.js 18+
- pnpm (`npm install -g pnpm`)

## Install Dependencies

```bash
cd shola-os
pnpm install
```

## Development

```bash
pnpm dev
```

Starts Vite dev server (usually http://localhost:5173)

## Build

```bash
pnpm build
```

Production build to `dist/`

## Preview Build

```bash
pnpm serve
```

Preview production build locally

## Environment

No special env vars required. Project uses default Vite config.
